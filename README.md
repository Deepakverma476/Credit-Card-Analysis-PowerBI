## Process
- Verified dataset for missing values, inconsistencies, and duplicates
- Cleaned and transformed data in Power Query
- Ensured all fields had the correct data types and formatting
- Conducted an initial structural review to understand key relationships
- Designed a Star Schema data model with fact and dimension tables
- Created DAX measures for revenue, amount, interest, trends, and segmentation
- Built multiple visuals as per project KPIs
- Developed two interactive dashboards:
     - Transaction Dashboard
     - Customer Dashboard
- Added slicers for week, gender, income level, card type, payment mode, etc
- Designed and formatted a professional visual experience with dynamic filtering

## Business Questions (KPIs)
### Overall Performance
- What are the total revenue, total amount, interest earned, and transaction count?
- How does weekly and quarterly revenue trend over time?
### Customer Analysis
- Which age groups, genders, income levels, and education levels generate the most revenue?
- Which states contribute the highest revenue?
### Card & Transaction Patterns
- Which card categories perform best?
- Which payment modes (Swipe/Chip/Online) generate the most revenue?
- What are the top revenue-generating expenditure types?
- Which customer job types contribute the most?
### Demographic Breakdown
- What is the revenue share by income type (High/Mid/Low)?
- How does dependent count affect spending behavior?
### Behavioral Insights
- Who are the top contributing customer groups?
- How do spending patterns differ across states, demographics, and transaction categories?
